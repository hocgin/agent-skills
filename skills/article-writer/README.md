# Article Writer Skill

AI驱动的智能写作系统，专注于创作高质量、低AI检测率的文章内容。

## 核心特性

- **快速写作模式**：AI自动生成初稿，无需人工介入
- **智能信息调研**：支持网页搜索、文档爬取、PDF提取
- **三遍审校机制**：系统化降低AI味，目标检测率 < 25%
- **通用文章类型**：支持自媒体、技术博客、正式文档等

## 快速开始

### 完整写作流程

**核心流程**（简化版）：

1. **信息调研** - 调用 `rules/research.md`
   - 支持文档URL自动爬取
   - 支持PDF提取
   - 支持Web搜索调研

2. **快速写作** - 调用 `rules/write.md`
   - 基于调研结果生成初稿
   - 支持3轮迭代修改
   - 口语化表达，降低AI味

3. **三遍审校** - 调用 `rules/review.md`
   - 第一遍：内容审校（事实、逻辑）
   - 第二遍：风格审校（降AI味）
   - 第三遍：细节审校（格式、标点）

4. **配图建议** - 调用 `rules/image.md`
   - 分析内容，生成3-5张配图方案
   - 支持 wenyan-cli AI 生图

**完整11步流程**详见 SKILL.md：

1. 定义需求 → 2. 选题讨论 → 3. 信息调研 → 4. 素材收集（可选） → 5. 经历提取（可选） → 6. 快速写作 → 7. 真实性检查 → 8. 三遍审校 → 9. 配图建议 → 10. 发布检查 → 11. 发布指南

### 单独使用规则

每个规则也可以单独调用：

- **只做调研**：使用 `rules/research.md`
- **只写作**：使用 `rules/write.md`（需要已有调研）
- **只审校**：使用 `rules/review.md`（需要已有草稿）
- **只配图**：使用 `rules/image.md`（需要已有文章）

## 与参考项目差异

相比 [article-writer-cn](https://github.com/wordflowlab/article-writer)：

1. **简化工作区系统** - 只保留通用工作区
2. **专注快速模式** - 以无人工介入的快速模式为主
3. **简化素材系统** - 素材收集为可选功能
4. **保留核心功能** - 调研、写作、审校、配图完整保留
