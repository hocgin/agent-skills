# 配图建议规则

> **目标**：分析文章内容，生成3-5张配图方案，提升文章可读性和吸引力

## 核心原则

- **基于文章内容**：配图应与文章主题和内容紧密相关
- **多样化类型**：封面图、对比图、流程图、截图等
- **具体化建议**：提供详细的配图描述和设计思路
- **实用导向**：考虑实际制作难度，推荐可行方案

---

## 使用方式

### 方式1：指定文章文件

```markdown
/images draft.md
```

### 方式2：自动推断

```markdown
/images
```

AI会自动查找最新的draft.md文件。

## 处理流程

### 1️⃣ 读取文章

**AI操作**：
- 读取draft.md全文
- 提取关键信息：
  - 文章标题
  - 核心主题
  - 主要章节
  - 关键数据和对比
  - 技术细节

### 2️⃣ 分析配图需求

**分析维度**：

#### A. 文章类型

| 文章类型 | 配图重点 | 示例 |
|---------|---------|------|
| 产品评测 | 对比图、截图、数据图 | Claude Code vs Cursor 功能对比 |
| 技术教程 | 截图、流程图、代码示例 | 如何配置Claude Code |
| 观点文章 | 抽象概念图、引用图 | AI编程的未来趋势 |
| 经验分享 | 场景图、时间线图 | 我的学习路线图 |

#### B. 章节结构

- 开头是否需要吸引眼球的封面图？
- 中间是否有需要可视化的数据对比？
- 是否有复杂流程需要流程图？
- 是否有技术细节需要截图说明？

#### C. 目标平台

- **公众号**：建议尺寸900×500px，风格简洁
- **技术博客**：可以使用详细的技术图表
- **通用**：平衡美观和信息量

## 配图设计原则

### 1. 风格统一

- 所有配图使用相同的配色方案
- 统一字体和字号
- 统一边框和阴影效果

### 2. 信息清晰

- 一张图只传达一个核心信息
- 避免信息过载
- 文字大小适中，确保可读

### 3. 美观实用

- 符合目标平台的审美
- 突出重点信息
- 适当的留白

### 4. 版权合规

- 使用免费可商用图片
- 或使用自己制作的图片
- 标注图片来源（如需要）

## 常见问题

### Q1: wenyan-cli 需要什么配置?

**A**:
1. **安装**: `pnpm install -g @hocgin/wenyan-cli`
2. **获取七牛云 Token**:
   - 访问七牛云控制台: https://portal.qiniu.com
   - 开通 AI 生图服务（Kling AI）
   - 获取 API Token
3. **执行命令时**:
   - 指定 `--service qiniu`
   - 指定 `--model kling-v1-5`
   - 指定 `--token <your_token>`
4. 支持中文提示词，效果更好

**示例配置**:
```bash
# 设置环境变量（推荐）
export QINIU_AI_TOKEN="your_token_here"

# 使用时直接引用
wenyan image \
  --service qiniu \
  --model kling-v1-5 \
  --token $QINIU_AI_TOKEN \
  --prompt "科技感AI编程助手" \
  --path ./output.png
```

### Q2: wenyan-cli 生图需要多久?

**A**:
- 单张图片生成时间: 2-3分钟
- 建议设置 `--timeout 180`（3分钟超时）
- 可以批量生成多个命令，同时执行

### Q3: 如何写好 wenyan-cli 的提示词?

**A**:
提示词格式: `主体描述 + 风格 + 色调 + 质量`

**示例**:
- ❌ 差: "猫"
- ✅ 好: "一只可爱的橘猫坐在窗台上，阳光明媚，摄影风格，高清"
- ❌ 差: "科技"
- ✅ 好: "未来科技城市，蓝色霓虹灯，赛博朋克风格，8k高清"

**技巧**:
- 具体描述主体（谁/什么）
- 添加风格词（简约/科技/插画等）
- 指定色调（蓝色/暖色/黑白等）
- 强调质量（高清/4k/8k/精细）

### Q4: 没有设计基础，如何快速制作配图?

**A**:
1. **推荐**: 使用 wenyan-cli AI 生图（5分钟）
2. 或使用 Canva/创客贴的模板（15分钟）
3. 只需替换文字和配色

### Q5: 配图数量多少合适?

**A**:
- **最少**: 1张封面图
- **推荐**: 3-5张（封面 + 数据图 + 2-3张内容图）
- **最多**: 不超过8张（避免喧宾夺主）

### Q6: 配图尺寸有什么要求?

**A**:
- **公众号**: 900×500px（横图），1080×1080px（方图）
- **技术博客**: 1200×630px（推荐）
- **社交媒体**: 1080×1080px（Instagram），1200×675px（Twitter）

**wenyan-cli 支持的宽高比**:
- `16:9` - 横版（封面、横图）
- `1:1` - 方形（社交媒体）
- `9:16` - 竖版（手机壁纸）
- `4:3` - 传统比例
- `3:4` - 竖版传统比例
