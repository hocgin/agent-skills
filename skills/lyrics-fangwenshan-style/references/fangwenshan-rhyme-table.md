# 方文山韵脚表整理版

这份表是依据当前目录内的课程字幕整理出的实战版摘要，重点是“听觉韵”，不是严格的语音学分类。

主要参考：

- `transcripts/第06天-主、副歌与韵脚使用之_哔哩哔哩_bilibili_BV1ZrAmecEVU_字幕.srt`
- `transcripts/第10天-歌词词汇之积累与收集_哔哩哔哩_bilibili_BV1ZrAmecEVU_字幕.srt`

说明：

1. 字幕有转写误差，下面的韵组名称做了便于写作的整理。
2. 重点是“听起来能不能押”，不是拼音尾巴必须完全一样。
3. 每次写一段前，先选一个韵组，再为这一组收集 8 到 20 个可用尾字或尾词。
4. 罕见字可以少量使用，但不要为了稀有而生硬。

## 13 类听觉韵

| 韵组 | 听感归并 | 常见尾字或尾词示例 | 使用提示 |
| --- | --- | --- | --- |
| 之思韵 | `-i`（`zhi/chi/shi/ri/zi/ci/si` 一类听感） | 心事、城市、样子、往事、位置、名字 | 适合口语句尾，顺口但容易太平，要靠画面提质。 |
| 阿韵 | `a` | 说话、想家、脸颊、落下、玫瑰花、晚霞 | 适合抒情和开阔画面。 |
| 翁韵 | `ong` | 霓虹、面容、天空、惶恐、时钟、汹涌 | 听感圆润，适合夜景、都市感。 |
| 欧窝韵 | `ou/uo` | 回首、回头、难过、轮廓、烟火、承诺 | 很常用，适合叙事与回忆。 |
| 欸耶韵 | `ei/ie/ue/ui` 的近似听感归并 | 心碎、憔悴、感觉、麻雀、多嘴、风雪、蝴蝶 | 这是整理性命名，核心是顺耳，不必死扣拼音。 |
| 衣鱼韵 | `i/ü` 的近似听感归并 | 回忆、泪滴、雏菊、结局、秘密、距离、呼吸 | 适合细腻抒情，也常用于副歌。 |
| 乌韵 | `u` | 孤独、幸福、道路、征服、落幕、归途、包袱 | 方文山在字幕中专门提到可多收集罕见词。 |
| 开韵 | `ai` | 明白、坦白、分开、重来、花开、等待、窗台 | 直给、流行感强，副歌易记。 |
| 鹅韵 | `e` | 坎坷、黄河、角色、隔着、折射、舍得 | 适合叙事句尾，注意避免过多口水词。 |
| 安昂韵 | `an/ang` 的近似听感归并 | 沮丧、脸庞、河岸、沙滩、港湾、风霜、月光 | 适合做大开口韵，画面张力比较足。 |
| 恩韵 | `en` | 纷纷、草木深、一个人、人事易分、黄昏、余温 | 易出古风与叙事情绪，但要防止句子太旧。 |
| 凹韵 | `ao` | 微笑、你好、海角、饮料、燃烧、拥抱、喧嚣 | 明亮、外放，适合副歌和流行口吻。 |
| 因音韵 | `in/ing` 的近似听感归并 | 身影、声音、风景、倒影、脚印、清醒、约定 | 适合轻快或悬停感较强的旋律。 |

## 实战用法

### 1. 先选段落韵组

常见做法：

- 主歌一个韵组，副歌一个韵组
- 四行里前两行一个韵，后两行一个韵
- 整段双数句押韵，单数句负责叙事

### 2. 先做尾字库

不要一边想故事一边硬找押韵字，先做一小组尾字库，例如：

- 安昂韵：港湾、河岸、脸庞、风霜、月光、纸船、摇晃、回廊
- 乌韵：落幕、归途、包袱、孤独、顽固、荒芜、雾、树

### 3. 句尾优先，句中次之

流行歌词最重要的是句尾听感。
句中如果刚好能形成内韵更好，但不要为了内韵牺牲句意。

### 4. 不为韵而韵

字幕里反复强调两件事：

- 不能为了用韵而用韵
- 不能为了换字而换字

判断标准很简单：

- 这个尾字是不是自然会出现在这个场景里
- 换掉它之后，整句意思会不会更顺

如果答案是否定的，就不要硬押。

## 常见段落模板

### 模板 A：双数句押同韵

第 1 行：起画面  
第 2 行：落韵  
第 3 行：补动作或信息  
第 4 行：再落同韵

### 模板 B：主歌稳，副歌亮

主歌：用叙事韵，如欧窝韵、恩韵、安昂韵  
副歌：用记忆点更强的韵，如开韵、凹韵、衣鱼韵

### 模板 C：先词库，后整段

1. 先列 10 个同韵尾词  
2. 再列 6 个场景名词  
3. 再列 6 个动作词  
4. 最后组合成句

## 额外提醒

这份整理版更适合“创作流程”而不是“语言学考据”。
当某个字放在旋律里听感顺耳，就可以优先按听觉韵处理。
